=== KNOWN PATTERNS ===:
AUTH FLOW: Login page posts credentials to /auth/login, then stores token and role in localStorage; logout clears token and role, and optional /auth/logout call is made.
PROTECTED ROUTE PATTERN: src/components/ProtectedRoute.tsx checks localStorage token and redirects to /login when missing; protected pages are nested under Route element ProtectedRoute in src/App.tsx.
ERROR HANDLING PATTERN: try/catch blocks with toast.error for UI feedback and fallbacks such as redirecting to /login on auth failures.
DATA FETCHING PATTERN: Axios-based calls to API_URL endpoints with Bearer token headers from localStorage for protected endpoints; helper modules under src/api encapsulate common calls.
